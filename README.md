Даалгавар 4: Quiz тоглоом
Гүйцэтгэлийн шаардлагууд
Асуултыг зохион байгуулах логик
Асуулт-хариултын тоглоомын гол хөдөлгүүр бол мэдээж асуултууд. Энгийн нэг хувилбарт би доорх гурван элемент бүхий асуултыг зураглая: “question” текст, “choices” гэх дөрвөн сонголтын жагсаалт, “answer” буюу зөв хариулт. Хэрхэн үүнийг Python-д дүрслэх вэ? Таны толгойд төрөл бүрийн шийдэл орж ирж байгаа байх; хамгийн түгээмэл нь dictionary. Dictionary ашигласнаар “асуулт” гэдэг түлхүүрт текстээ хадгалах, “сонголтууд” гэдэг түлхүүрт list хадгалах, “хариу” түлхүүрт зөв сонголт хадгалах шийдэл тун ойлгомжтой.

Жишээлбэл, нэг асуулт маань ингэж харагдана:

q1 = {
    "question": "Эсрэг цэнэгтэй бөөмийн нэр юу вэ?",
    "choices": ["Протон", "Электрон", "Нейтрон", "Фотон"],
    "answer": "Электрон"
}

Хэрвээ бид хорин тав, тавин асуулттай өргөн хүрээний Quiz хийх бол олон dictionary-г нэг list дотор цувуулж хадгалах, эсвэл бүр JSON файл дотор хадгалаад хожим унших арга бий. Оноо тооцохдоо бид хэрэглэгчийн оруулсан “B” эсвэл “2” гэх мэт индекс дээр тулгуурлан зөв буруу эсэхийг шалгана. Dictionary-н давуу тал нь question["answer"] гэж л шалгахад хангалттай энгийн юм.

Файл дээр суурилсан асуултаа бэлтгэх
Эхнийх нь CSV (comma-separated values) файл. Хоёр дахь нь JSON. CSV-гийн хувьд Excel эсвэл Google Sheets-ээс шууд экспортлох боломжтойн дээр, хамгийн энгийн текст хэлбэрийн хүснэгт тул асуудалгүй. Харин JSON нь Python-д json модулиар шууд dict хэлбэрт хувирч орж ирдэг учраас код бичих талдаа илүү хялбар. Аль алинд нь бид богино жишээ бичье.

CSV-гийн нэг мөр дараах бүтэцтэй байлаа гэж төсөөлье:

question,choice1,choice2,choice3,choice4,answer

Тэгвэл Python талдаа бид csv модулийг импортлоод:

import csv

questions = []

with open("quiz.csv", encoding="utf-8") as f:

    reader = csv.DictReader(f)

    for row in reader:

        q = {

            "question": row["question"],

            "choices": [row["choice1"], row["choice2"], row["choice3"], row["choice4"]],

            "answer": row["answer"]

        }

        questions.append(q)
Ингээд questions list-д маань олон dictionary цугларна. Юу ч хүндрэхгүй.
